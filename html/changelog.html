29 March 2021

ProbablyCarl updated:

  • Returns functionality of the Captain's sidearm, once again locking it to him alone and upping the damage to original values.
  • Augmentations have been returned, though given appropriate costs for what their functionality is.
  • To make exploration more independent, alongside giving them more utility, they've two area locked laser carbines, which are functionally identical to the Infantry Laser Carbine. The carbines are located inside the 'boomstick' locker, next to the Pilot's cabin.
  • Pathfinder's hardsuit returned to old values for ballistic protection, making it anything but the plastic suit it was turned into. Given it's an Exosuit, you'd expect it to be semi-decent.
  • Lawyers, Bailiff and Adjudicator has been removed. See the maptweaks for more information. We're in space, not on a colony, so frontier justice is to be expected.
  • Research now has, inside the maintenance tunnels west, a long term storage for biologicals and artifacts. This is to primarily be used by Expeditionary groups, so to fulfill this it permits them access. This replaces the courthouse.
  • Lawyer office next to cryo is now an unused location, free to be used by the crew.
  • Adjudicator's office is now a VIP Quarters.
  • Infantry returned. Their prep is now south of where it was, using an area of the old ship garden.
  • 'Subdural' armor is now 'Subdermal' armor.
  • Returns the one-shot capability of the Alate's Defense Lance.
  • Various melee weapons have had their parry chance re-added.
  • 7mm cartridge given five armor pen, so it has at least some guaranteed damage. If this turns out to be a change for the worse, we'll revert it.
  • Introduction sound changed. If this is too gloomy, it will be reverted.
  • Laser Carbine sound changed, so it doesn't sound like a sausage being smacked against a wet deck.
  • Probably something important.
